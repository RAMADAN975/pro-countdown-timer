# ⏳ Advanced Countdown Timer - v6.0

A high-performance, feature-rich countdown timer application built with **React** and **Vite**. This project demonstrates advanced React concepts like custom hooks logic, persistent storage, and real-time API integration.

## 🚀 Key Features
* **Precision Control:** Dual input fields for both Minutes and Seconds.
* **Dynamic UI:** Interactive progress bar that changes color (Green to Red) during the final 10 seconds.
* **Session Persistence:** Automatically saves finished countdowns to **LocalStorage**.
* **Motivation Engine:** Fetches random inspirational quotes using the **Quotable API**.
* **Full Controls:** Start, Pause, Resume, and Reset functionality with clean state transitions.
* **Responsive Design:** Fully optimized for mobile and desktop using **Bootstrap 5**.

## 🛠 Tech Stack
* **Frontend:** React.js (Hooks: `useState`, `useEffect`, `useRef`)
* **Build Tool:** Vite (Ultra-fast development)
* **Styling:** Bootstrap 5 & CSS3
* **Data:** LocalStorage API & Fetch API
